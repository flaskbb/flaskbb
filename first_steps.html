First Steps

Getting set up with FlaskBB is easy, just follow these few short steps and you will have a fully functional bulletin board.

Installation

To install FlaskBB, you will first need a server capable of running a webserver.

Dependencies

Before you can download and install FlaskBB you will need a few dependencies on your server first:

  • Python 2.7+
  • pip
  • git

Install Virtualenv

To make it easier for you to test FlaskBB without interfering with your main python packages, you can use virtualenv.

Virtualenv allows you to create virtual python environments, that you are free install any python packages you wish.

pip install virtualenv

Create Virtualenv for FlaskBB

We will want to create a virtualenv for flaskbb

virtualenv flaskbb

We will then want to activate your new virtualenv so that any dependencies you install with not interfere with anything else on your machine.

source flaskbb/bin/activate

Get FlaskBB and install dependencies

Now we will want to download FlaskBB from github and install its dependencies.

cd flaskbb
git clone https://github.com/sh4nks/flaskbb.git

Then we will install the dependencies on your new virtualenv.

cd flaskbb pip install -r requirements.txt

Configuration

The next thing we will need to do is configure the FlaskBB

Doing so is easy, just activate the developement.py.example and create the test configuration

cp flaskbb/configs/development.py.example flaskbb/configs/development.py python manage.py createall

Run the server

Now you can start the server and start playing!

python manage.py runserver

This will create and instance of FlaskBB, running on http://localhost:8080
