Features

FlaskBB comes with a lot of inbuilt features, such as private messages, a topic tracker, a unread/read tracker, a own permissions system (!) and last but not least with a simple admin interface.

A Bulletin Board like FluxBB or DjangoBB in Flask.

A minimalist Bulletin Board written in Flask, the microframework. Resulting in a fast, light and easy to use bulletin board.

Private Messaging. Communication made easy.

With private messaging built into FlaskBB, it is easier to for your users to communicate amongst each other

Administrator interface. It'll blow your mind.

We try to keep the administrator interface as clean and simple as possible. Nobody should have to learn how to administrate a forum.

Permission System. See for yourself, if you have the permission.

Another nice feature that we provide, is a permission system. For example, I you want to have a special group with special permissions - just create one - it is possible!

Moderation tools. Easy accessible

Do you ever wanted to easily moderate a forum? No problem - now you can! Simply click on a topic (you need to have the right permissions of course) and do what your heart desires.

BBCode support. Easy markup

With support for BBCode you are able to format text, insert links, create tables and add emoticons to improve the users experience.

Topic Tracker. Get the latest updates from your favourite topics.

The Topic Tracker helps you stay on top of all the latest posted to your favourite topics. All in an easy to use interface

Unread topics and forums. Quick overview of unread topics

With a visual distinction between read topics and unread topics, there is no need to read the same topic twice
